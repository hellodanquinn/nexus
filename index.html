To enable the live chat

The chat screen calls a serverless function that proxies to the Anthropic API. Your API key lives in Netlify's environment variables — never in the browser. Set it once and the chat works immediately.

1

Get your Anthropic API key

Visit console.anthropic.com → API Keys → Create key

2

Add it to Netlify

Site settings → Environment variables → Add variable
Key: ANTHROPIC_API_KEY   Value: sk-ant-...

3

Trigger a redeploy

Deploys → Trigger deploy → Deploy site. Takes about 30 seconds.

4

Open the chat and type a query

Try: Who should I reconnect with this week?
